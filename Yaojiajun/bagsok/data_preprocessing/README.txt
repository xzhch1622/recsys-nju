这个文件夹下面是关于keyword preprocessing的代码。

class.stemmer.inc.php 和 porter_stemmer.php是网上关于抽取词根的poter stemmer算法的PHP实现，想看看是否用的上

data_preprocessing.php 是我写的预处理程序，基本完成了。

extract_keywords.php 是玮哥的提关键字的程序，拿来用用

extract_keywords_to_table.php 是提取关键字并将它们用空格分开，放入表keywords_from_userinfo里面，方便预处理

create_table_keyword.sql 是建表keyword的脚本
